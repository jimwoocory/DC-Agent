## 拉群流程速查

**T-1 天 / 30 分钟**：用 [私聊预告模板.md](私聊预告模板.md) 私聊每个要拉的员工。

**T 时刻 · 拉群**：
1. 飞书新建群 **「DC-Agent · AI 小助手内测群」**
2. 拉 5-10 个核心员工 + 巅池-Agent 小助手
3. 第一条消息发 [飞书卡片_欢迎.json](飞书卡片_欢迎.json) 的内容（卡片格式）
4. 群里同步发 [飞书卡片_部门培训.json](飞书卡片_部门培训.json)，让员工先找到自己部门怎么用
5. 发 [飞书卡片_部门测试任务.json](飞书卡片_部门测试任务.json)，让员工完成培训后小测试；员工按格式回复后小助手自动批改
6. 群里同步发 [员工使用指南.html](员工使用指南.html) 的链接/附件
7. 小测试通过后，按 [破冰任务清单.md](破冰任务清单.md) @ 各部门员工试第一个真实工作任务

**T+5 分钟**：观察员工试用反应，看有没有冷场 / 异常

**每天下班前**：按 [反馈机制.md](反馈机制.md) 收一轮反馈
