# 破冰任务清单 · 按部门定制

> **用途**：拉群后 5-10 分钟内，给每个员工分配一个"开胃菜"任务
> **目的**：避免冷场 + 让员工立刻感受到 AI 助手的实际价值
> **使用方式**：你直接 @ 对应员工 + 抛出这个任务，让他们试

---

## 1️⃣ 市场部（罗鸣 / 杜云婷）

### 推荐任务 A · 客户问候话术
```
@罗鸣 @杜云婷 来试试 ——

@巅池-Agent小助手 帮我写一条端午节客户微信问候话术，对象是公司 VIP 客户，语气要专业又有人情味，结尾推荐一个新方案
```

**期望效果**：30 秒内出一份可直接发到微信/私域/飞书的客户触达话术，结构完整（称呼 + 节日祝福 + 业务推进 + 署名）。

### 推荐任务 B · 客户跟进话术
```
@巅池-Agent小助手 我下午要跟一个犹豫 2 周的客户复联，他之前说"再考虑下"，帮我准备 3 套不同语气的开场白
```

## 进群后的"开场白"模板

把以下消息复制到群里，启动整个测试：

```
@全体成员 各位下午好～AI 小助手内测群正式启动 🎉

群规请看上面的卡片，重点 3 条：
  1. 不在群里讨论敏感数据
  2. 关键决策人工二次确认
  3. 体验反馈直接发我

我给每个部门准备了 1-2 个"开胃菜"任务，可以照着试一下。也欢迎你**抛自己工作中的真实问题**给小助手试。

我会全程在线，出问题立刻处理。开始吧 👇
```

然后立刻 @ 几个核心员工，丢出对应的开胃菜任务，让群活起来。

---

## 注意事项

- **不要全员一起 @**，分批次（每 5 分钟 @ 1-2 个），避免群刷屏
- 老板的任务**让他自己抛出来**（你不要替他发），他会觉得"被尊重"
- 你自己也要试 1-2 个任务（让员工觉得你也在"玩"，不是高高在上的产品经理）
