# 部门培训小测试答案

> 用途：小助手会自动批改「飞书卡片_部门测试任务.json」里的培训后小测试；本文件作为管理员备用答案和复核模板。

## 标准答案

| 题号 | 答案 | 批改说明 |
|---|---|---|
| 1 | B | 群里不能发客户名单、合同金额、员工隐私、财务明细。 |
| 2 | C | 好问题要包含任务、对象、语气、限制和输出要求。 |
| 3 | B | 涉及公司真实信息时，需要提供可核验来源。 |
| 4 | A | 先用小助手处理本部门的草稿、整理、拆解、复盘类工作。 |
| 5 | A | 对外内容必须人工复核后再使用。 |

## 通过标准

- 5 题答对 4 题即可通过。
- 第 4 题只看员工所在部门那一行，正确答案都是 A。
- 如果员工第 1、3、5 题任意一题答错，建议重点提醒安全边界和人工复核。

## 自动批改说明

员工按下面格式在群里回复时，小助手会自动返回分数、错题和下一步：

```text
部门：市场部；答案：1B 2C 3B 4A 5A；一句话反馈：我最想用它帮我 ...
```

也支持紧凑格式：

```text
部门：市场部；答案：BCBAA
```

## 人工批改反馈模板

```text
收到，你这次小测试答对 X/5，已通过。

提醒一句：涉及客户、合同、财务、员工隐私的信息不要发群里；对外内容一定要人工复核。

接下来可以做破冰任务：用你今天真实工作里的一件事 @ 小助手试一次。
```

```text
收到，你这次小测试答对 X/5，还差一点。

建议再看一遍培训卡里的“安全边界”和“怎么提问”。尤其记住：公司真实信息要给来源，对外内容要人工复核。
```
